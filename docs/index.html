💎 Sapphire

Open source framework for turning an AI into a persistent being. 11 personas, voice, memory, tools, stories. Self-hosted, privacy-first.

What Makes Sapphire Different

🎭 11 Personas

Each persona bundles a prompt, voice, toolset, and personality. Switch persona = switch everything. Build your own.

🎤 True Voice Assistant

Wake word detection, speech-to-text, text-to-speech. Hands-free operation with any mic and speaker.

🧠 Multi-LLM Support

LM Studio, llama.cpp, Claude, GPT, Fireworks, any OpenAI-compatible endpoint. Switch models per conversation.

🛠️ 65+ Tools & Tool Maker

Web search, memory, notes, image generation. The AI writes, validates, and installs new tools at runtime.

✏️ Assembled Prompts

Modular prompts with swappable pieces. Spice injection keeps responses fresh. AI can edit its own prompt mid-conversation.

📚 Memory & Knowledge

Semantic vector search across memories, organized knowledge base with file upload, contact book, and hierarchical goals.

⏰ Heartbeat

Scheduled autonomous tasks. Morning greetings, dream mode, alarm clock, random hellos. Cron timing with probability gates.

🎲 Story Engine

Built-in game master. Dice rolls, branching choices, progressive clues, room navigation. Run tabletop RPGs from a JSON preset.

🏠 Home Assistant

Smart home control via voice or text. Lights, scenes, thermostats, switches, phone notifications.

🔒 Privacy-First

One toggle blocks all cloud connections. Private prompts only activate locally. No telemetry. Fully self-hosted.

🔌 Plugin Architecture

Keyword-triggered Python extensions and JavaScript web UI plugins. Extend both backend and frontend.

🖼️ Image Generation

SDXL with character replacement for visual consistency across scenes. Gallery displayed inline in chat.

Quick Start

# Linux only
sudo apt-get install libportaudio2

# Create environment
conda create -n sapphire python=3.11 -y
conda activate sapphire

# Clone and run
git clone https://github.com/ddxfish/sapphire.git
cd sapphire
pip install -r requirements.txt
python main.py

# Web UI at https://localhost:8073

Requires Python 3.11+, 16GB+ RAM. Nvidia GPU recommended for TTS/STT. Local LLM via LM Studio or cloud via Claude/GPT.
